# Projekt: DNA → mRNA → Aminokwasy

## Autor
Maja Przyłęcka

## Data oddania
22.06.2026

## Opis projektu
Program wczytuje sekwencję DNA z pliku tekstowego,
następnie:
- zamienia DNA na mRNA
- wyszukuje kodon START (AUG)
- tłumaczy sekwencję na aminokwasy
- zatrzymuje się na kodonie STOP
- wypisuje listę aminokwasów oraz ich częstość występowania

## Technologie
C++ (CLion)
